# Pages

## `/`
- 홈 화면
- 모든 주요 페이지로 바로 이동할 수 있는 공통 내비게이션 제공

## `/dart`
- DART 금일 호재 공시 기본 화면
- 오늘 기준 호재 공시를 최신순 또는 분류별로 확인
- 푸시 알림 전체/DART/SEC 토글 제공

## `/dart/opendart-fast`
- OPEN DART 공시검색 API 기반 빠른 공시 화면
- 오늘 공시를 공식 API로 직접 조회
- 주요사항보고, 기타공시, 거래소공시를 기준으로 강한 호재를 추출
- KOSPI, KOSDAQ만 대상으로 무료 환경에서 동작

## `/sec`
- SEC 금일 호재 공시 기본 화면
- 오늘 기준 호재 공시를 최신순 또는 분류별로 확인
- 푸시 알림 전체/DART/SEC 토글 제공
