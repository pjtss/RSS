# Pages

## `/`
- 홈 화면
- DART, 국내 주식 급속 호재, OPEN DART 빠른 공시, SEC 페이지로 이동하는 진입점

## `/dart`
- DART 금일 호재 공시 기본 화면
- 오늘 기준 호재 공시를 최신순 또는 분류별로 확인

## `/dart/rapid`
- 국내 주식 급속 호재 전용 화면
- DART RSS 데이터를 5초 주기로 다시 확인
- 최강호재 우선 정렬
- 최근 5분 내 공시와 주요 키워드를 빠르게 확인

## `/dart/opendart-fast`
- OPEN DART 공시검색 API 기반 빠른 공시 화면
- 오늘 공시를 공식 API로 직접 조회
- 주요사항보고, 기타공시, 거래소공시를 기준으로 강한 호재를 추출
- KOSPI, KOSDAQ만 대상으로 무료 환경에서 동작

## `/sec`
- SEC 금일 호재 공시 기본 화면
- 오늘 기준 호재 공시를 최신순 또는 분류별로 확인
